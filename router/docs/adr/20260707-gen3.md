## Context and Problem Statement

X2 Gen3.0 車両の車外通信インフラとして使用しているセルラールータについて、現行リファレンスモデル HYTECINTER HW5G-3200-V2(N) に複数の運用上の問題（特定電波状況下でのプロセス再起動による通信途絶、SIM認識不良、SSH経由のRSSI取得時クラッシュ）が発生しており、かつ近々EoLを迎えるため追加調達が困難となる。Gen3.0向けに信頼性の高い代替ルータを選定する必要がある。

## Decision Drivers <!-- optional -->

- 遠隔監視（WebRTC映像配信）・緊急停止介入に耐えうる通信安定性（WAN UL 平均10-20 Mbps）
- マルチキャリア対応（NTT/KDDI/Softbank、KDDI必須）
- 車載グレードの環境耐性（IEC 60068等の環境テスト合格、動作温度 0-45℃以上）
- 技適取得済み（グローバル展開を見据えCE/FCC/UKCA対応が望ましい）
- 電波強度・周波数帯のSW的取得可能性（運用監視・サイトサーベイへの活用）
- EoLリスク回避（長期供給保証）
- 仕向地ごとに電波状況が異なるため、単一機種に限定せず最適な組み合わせを選択可能とする柔軟性
- 今後強まるCyber Security要件（UN R-155/R-156）への対応余地
- その他、細かい要求仕様については以下を参照
  - https://tier4.atlassian.net/wiki/spaces/NGSS1st/pages/3639837906/memo

## Decision Outcome

Chosen option: "Teltonika RUTM50、Robustel R5020-Lite、Peplink MAX BR1 mini 5G の3機種を詳細評価対象として選定する"
11機種の初期候補に対し、Decision Drivers に基づくスクリーニングを実施した結果、以下の必須条件を全て満たす3機種を上位候補として選定し、フィールドサーベイ（走行評価）による詳細比較に進める。

スクリーニング条件（全て必須）：

- 5G/LTE対応
- 技適取得済み
- 車載グレード環境テスト合格（IEC 60068またはE-mark）
- EoL予定なし（長期供給見込み）
- 代理店サポートが有効かつ迅速
- グローバル仕向に対応可能（CE/FCC/UKCA等の認証取得）

なお、上位3機種は全て同一の通信モジュール（Qualcomm RM520N-GL）を搭載しており、最終的な性能差はルータ側のソフトウェア処理・アンテナ設計・熱設計に起因する。

### Positive Consequences <!-- optional -->

- 現行V2の既知問題（通信途絶、SIM認識不良、クラッシュ）からの脱却
- 3機種とも異なる強みを持ち、仕向地・用途に応じた柔軟な選択肢を維持できる
- 同一通信モジュール搭載のため、評価結果の比較が公平に行える
- いずれの機種もEoLリスクがなく、長期的な調達安定性が見込める

### Negative Consequences <!-- optional -->

- 最終1機種の決定が未了であり、走行評価の完了まで確定できない
- 3機種の評価工数（機種 × キャリア × 仕向地）が発生する
- 評価期間中は暫定的に複数機種を並行管理する必要がある

## Pros and Cons of the Options <!-- optional -->

### Option 1: Teltonika RUTM50

価格: 136,000円 / 通信モジュール: RM520N-GL / 代理店: マクニカ

- Good, because IEC 60068（振動、衝撃、バンプ、湿度）準拠
- Good, because EoL予定なし、MTBF 384,109h（25℃）
- Good, because 入力電圧 9-50 VDC で車載環境に適合
- Good, because 動作温度 -40〜75℃ と広範囲
- Good, because バンド/CA切替が瞬時に行われる
- Good, because Gen2 V2 の外部アンテナ（YOZ-52544）を流用可能
- Good, because 保証期間2年（3年まで延長可能）、製品保証期間外も技術的な質問回答は受付可能
- Good, because 利用帯域指定可能
- Good, because UK向けに同系列機 RUTX50 が利用可能
- Bad, because リブート時に内部ログ消失（外部 syslog server 必須）
- Bad, because E-mark 未取得（同型機 ATRM50 では取得済み）
- Bad, because ISO 16750 未認証
- Bad, because 納期 6週〜3ヶ月（本国出荷）

### Option 2: Robustel R5020-Lite

価格: 96,000円 / 通信モジュール: RM520N-GL / 代理店: MCM Japan

- Good, because 候補3機種中で最も安価（96,000円）
- Good, because IEC 60068-2-6/64/27（振動・衝撃）準拠
- Good, because E-mark ECE R10 取得済み
- Good, because MTBF 332,445h（25℃）
- Good, because HORIBA MIRA 推奨機種であり、UK/AUS でも利用可能（CB、CE、UKCA、RCM、テレコ、JATE取得）
- Good, because 無料クラウド管理プラットフォーム RCMS 提供
- Good, because 保証期間2年
- Good, because 内部ストレージへのログ保存 + syslogd 対応
- Good, because 利用帯域指定可能
- Bad, because 動作温度 -20〜55℃（通常温度）と他候補より狭い
- Bad, because VRRP 機能は上流回線断時に切り替えが発生せず、自動運転車両の回線冗長化には不適
- Bad, because WiFi モジュールの技適が取れていない（無線モジュール自体は問題なし）

### Option 3: Peplink MAX BR1 mini 5G

価格: 190,000円 / 通信モジュール: RM520N-GL / 代理店: CASO

- Good, because E-Mark 取得済み（ECE R10）
- Good, because EN 45545-2（鉄道防火）、EN 61000（EMC）準拠
- Good, because 動作温度 -40〜65℃
- Good, because SpeedFusion VPN によるボンディング・冗長化機能
- Good, because InControl2 クラウド管理による遠隔設定・可視化が可能
- Good, because IDS機能あり（Cyber Security対応の足がかり）
- Good, because EoL予定なし、MTBF 2,369,129h（25℃）と非常に高い
- Good, because ローカル5G対応（NTT com との接続検証実績あり）
- Good, because SSH CLI で電波情報取得可能
- Good, because 利用帯域指定可能
- Bad, because 候補3機種中で最も高価（190,000円）
- Bad, because 保証期間1年（翌年以降は有償 Warranty）
- Bad, because SpeedFusion 利用時は対向ルータまたはクラウドプランが別途必要

## Next Action <!-- optional -->

- 上位3機種 × 対象キャリア（KDDI必須）のフィールドサーベイ（走行評価）を実施
- 仕向地別の機種選定方針の策定（UK向け RUTX50 / R5020 等の評価）
- Cyber Security 要件（IDS/IPS、L2/L3トンネリング）への対応方針検討

## Remarks <!-- optional -->

- 本ADRは「11機種 → 3機種」のスクリーニング結果を記録するものであり、最終的なリファレンス機の決定は走行評価後の別ADRで行う
- 仕向地によっては、本ADRで除外した機種が再検討対象となる可能性がある（例: UK向けに R5020 が有力）
- 3機種とも通信モジュールは同一（Qualcomm RM520N-GL）であるため、走行評価ではルータ固有の差異（SW処理、アンテナ設計、熱設計）が評価ポイントとなる
- NOKIA FRR0501c は技術的には有力だったが、日本窓口消滅・本社Sales無反応というサポートリスクにより除外した。今後窓口が復活した場合は再検討の余地がある
